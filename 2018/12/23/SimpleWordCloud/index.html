SimpleWordCloud

Lareina
2018-12-23

1. 什么是词云图(WordCloud)？
  ”词云”就是对文本中出现频率较高的”关键词”进行视觉可视化，形成”关键词云层”或”关键词渲染”，从而滤掉大量的文本信息，使得读者可以快速了解文本主旨。
2. 工具+依赖包+环境配置（基于Windows 10）
2.1 安装Python & Pip
  首先检查电脑是否安装了Python(解释器) 和Pip(通用的Python包管理工具，提供了对Python包的查找、下载、安装和卸载功能。cmd窗口中输入命令： python –version && pip –version
(1)不存在Python和Pip，嗯，那就装一个吧！
  下载地址 ：Python（python3.7.1) & Pip（pip-18.1）
  保存路径：./soft —-自定义
  安装Python的时候，注意修改路径和添加到本地电脑环境变量中；
  安装Pip，可以直接cmd窗口进入安装安装包目录输入命令：python -m pip install -U pip
(2)存在Python和Pip,如下图所示：

2.2 安装第三方依赖库

  • numpy    : pip install nump(Python的一种开源的数值计算扩展,可用来存储和处理大型阵)

  • matplotlib  : pip install matplotlib(Python的2D绘图库）

  • wordcloud  : pip install wordcloud(词云图库)

注意 1.cmd窗口下执行上述命令，并且按照顺序安装(涉及库依赖);
   2.涉及问题权限的话，命令中添加 --user 即可;
3. 程序源码（点击代码显示或隐藏）

  代码

test01.py

#!/usr/bin/env python

Image-colored wordcloud
=======================    
#导入必要的库
from os import path
from PIL import Image
import numpy as np
import matplotlib.pyplot as plt

from wordcloud import WordCloud, STOPWORDS, ImageColorGenerator

# Read the whole text.
text = open(r'D:\WorkSpace\Python\wordcloud\HarryPotter.txt').read()
test01.array = np.array(Image.open(r"D:\WorkSpace\Python\wordcloud\test01.jpg"))

# 设置停用词
stopwords = set(STOPWORDS)
stopwords.add("said")

# 你可以通过 mask 参数 来设置词云形状
wc = WordCloud(background_color="white", max_words=2000, mask=test01.array, stopwords=stopwords, max_font_size=40, random_state=42)
# generate word cloud
wc.generate(text)

# create coloring from image
image_colors = ImageColorGenerator(test01.array)

# show
# 在只设置mask的情况下,你将会得到一个拥有图片形状的词云
plt.imshow(wc, interpolation="bilinear")
plt.axis("off")
plt.figure()
# recolor wordcloud and show
# we could also give color_func=image_colors directly in the constructor
# 我们还可以直接在构造函数中直接给颜色
# 通过这种方式词云将会按照给定的图片颜色布局生成字体颜色策略
plt.imshow(wc.recolor(color_func=image_colors), interpolation="bilinear")
plt.axis("off")
plt.figure()
plt.imshow(test01.array, cmap=plt.cm.gray, interpolation="bilinear")
plt.axis("off")
plt.show()
